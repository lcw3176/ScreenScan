# ScreenScan
원격 모니터 스캔 프로그램

### 서버
* 클라이언트와 연결되면 유저 목록에 자동 추가됨
* 유저 목록 클릭 시 화면 스캔 시작

### 클라이언트
* 프로그램 실행 시 서버와 바로 연결
* 서버에서 연결을 종료하면 클라이언트 프로그램도 종료


### 2020.11.11 코드 리팩토링
* 기존에 이미지를 받아오기 위해 바이트 변수 크기를 너무 크게 선언한 것 수정
* 비동기 -> 동기 방식으로 수정
